# Création des utilisateurs

Créer les comptes admin et Propriétaire :

```bash
# Admin
php bin/console app:create-user admin@carrosserie-lino.fr VotreMotDePasse admin

# Propriétaire
php bin/console app:create-user proprietaire@carrosserie-lino.fr VotreMotDePasse proprietaire
```

Remplacez les emails et mots de passe par vos valeurs.
